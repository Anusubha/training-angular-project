1.Number Pipes:

  • Number (4.678329) in 1.5 Format : {{4.678329 | number:'1.5'}}
  • Number (14.67839) in 5.3-4 Format : {{14.67839| number:'5.3-4'}}
  • Number (98.123) in 3.4-5 Format : {{98.123 | number:'3.4-5'}}
  • Percent (1.0) : {{1.0 | percent}}
  • Percent (0.25) in 2.1 Format: {{0.25 | percent:'2.1'}}
  • Currency with default format (50) : {{50 | currency}}
  • Currency with specific format (50) : {{50 | currency:'INR'}}
  • Currency with currency code (50) : {{50 | currency:'INR':'code'}}
  • Currency with currency code (50) : {{50 | currency:'INR':'code':'2.1'}}
